# Module Specifications:

1. This module can charge and discharge Lithium batteries safely
2. Suitable for 18650 cells and other 3.7V batteries
3. Charging current – 1A (adjustable)
4. Input Voltage: 4.5V to 5.5V
5. Full charge voltage 4.2V
6. Protects battery from over charging and over discharging
7. No verse polarity protection
